# Install Polymer CLI
npm install -g polymer-cli
# Initialize your project from a template
-Create a new project folder to start from.
.. code:: bash

		 mkdir my-app
		 cd my-app

-Initialize your project with an app template.
.. code:: bash

		 polymer init

 #Serve your project
.. code:: bash

		polymer serve --open

		polymer build

		install firebase
